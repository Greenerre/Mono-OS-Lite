# Video Demo

Open `demo/mono-os-lite-demo.svg` in a browser to play the animated Mono OS Lite walkthrough.

The animation simulates:

- Human chat input
- Mock speech-to-text input
- Instruction layer policy packet
- Multi-agent routing
- L3 financial approval gate
- Memory and visual context indexing
- Goal coverage and audit trail

This is intentionally an animated SVG instead of a recorded emulator video because the local Android ATD emulator rendered black framebuffer screenshots in this environment while the app was still running and interactive. A physical Android device or full Google APIs emulator image should be used for final judge-facing screen recording.
